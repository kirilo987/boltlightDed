# LightningOnDeath

![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21-blue)
![Java](https://img.shields.io/badge/Java-17-orange)
![License](https://img.shields.io/badge/License-MIT-green)

**LightningOnDeath** is a lightweight plugin for Minecraft that causes lightning to appear at the point of death of the player. The plugin is designed for Paper 1.21 servers.

## Installation
1. Download the latest version of the plugin from the [Releases](https://github.com/yourusername/LightningOnDeath/releases) section.
2. Place the `.jar` file in the `plugins` folder of your Minecraft server.
3. Restart the server.
